[ 05 ] — FAQ

Frequently asked questions.

Everything you need to know before you launch.

What is Plexus?

Plexus is a static, terminal-themed AI chat client. It talks to models through OpenRouter and stores everything locally in your browser.

Does this landing page change the chat?

No. The chat interface is served from chat.html and is identical to the previous index.html experience.

Do I need an account?

No Plexus account. You only need an OpenRouter API key, which you paste into settings on first use.

Is my data private?

Yes. Chats stay in localStorage. You can optionally encrypt them with a passphrase. Nothing is sent to a Plexus server — there isn't one.

What models are supported?

Any model available on OpenRouter, including free options like openai/gpt-oss-120b:free. Switch models in settings anytime.

Can I use this on mobile?

Yes. Plexus is responsive and works on phones and tablets with the same terminal aesthetic.
